# 2. 函数拟合实验报告（ReLU 神经网络）

## 2.1 实验目标

本实验通过构建一个两层的 ReLU 全连接神经网络，对给定的一元函数进行拟合。实验流程包括：

1. 定义目标函数。
2. 在函数区间上采样构造训练集与测试集。
3. 使用训练集训练模型。
4. 使用测试集评估拟合效果。

---

## 2.2 函数定义

本次实验选用的目标函数为：$y=\sin(x)$。

在代码中定义如下：

```python
def taget_function(x):
	return torch.sin(x)
```

说明：输入为一维实数张量，输出为对应位置的正弦函数值。

---

## 2.3 数据采集与数据集划分

### 2.3.1 采样方式

在区间 $[-\pi,\pi]$ 上等间隔采样 100 个点，得到输入数据 $x$；再通过目标函数得到标签 $y$。

```python
x = torch.linspace(-np.pi, np.pi, 100).view(-1, 1)
y = taget_function(x)
```

### 2.3.2 训练集/测试集划分

按 8:2 比例划分训练集与测试集，使用随机打乱索引方式进行抽样。

- 训练集比例：80%
- 测试集比例：20%
- 划分目的：用训练集学习参数，用测试集验证泛化能力

---

## 2.4 模型描述

### 2.4.1 网络结构

模型采用两层全连接结构（1 个隐藏层），激活函数为 ReLU：

1. 输入层：维度为 1
2. 隐藏层：`Linear(1, 64)`
3. 激活层：`ReLU()`
4. 输出层：`Linear(64, 1)`

对应代码：

```python
class RELUNet(nn.Module):
	def __init__(self, hidden_size=64):
		super().__init__()
		self.hidden = nn.Linear(1, hidden_size)
		self.relu = nn.ReLU()
		self.predict = nn.Linear(hidden_size, 1)

	def forward(self, x):
		x = self.hidden(x)
		x = self.relu(x)
		x = self.predict(x)
		return x
```

### 2.4.2 损失函数与优化器

- 损失函数：均方误差（MSELoss）
- 优化器：Adam
- 学习率：0.01
- 训练轮数：100
